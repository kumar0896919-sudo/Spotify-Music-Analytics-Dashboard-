# Spotify-Music-Analytics-Dashboard-

I created an interactive Spotify Music Analytics Dashboard using Power BI to analyze global music trends, artist performance, album types, and listening patterns.

🔍 Project Overview

This dashboard provides a complete overview of:

Total songs, artists, and average duration

Song popularity trends

Songs by year, album type, and explicit vs non-explicit content

Top artists and most played tracks

Monthly average popularity

Total albums by month

Interactive song preview section (UI-based)

🛠 Tools & Skills Used

Power BI

DAX Measures

Data Modeling

ETL & Data Cleaning

Advanced Visualizations

⭐ Key Features

Modern Spotify-inspired dark theme UI

Dynamic slicers for month/quarter view

Drill-down interactions

Custom visuals for artists, songs, and albums

Attractive metrics cards for KPIs

Smooth navigation with buttons (Home, Overview, Artists, Songs)
